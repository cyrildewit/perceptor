# Perceptor

> Modern Sass starter kit that requires you to keep your CSS organized and clean.

## Note

This project is currently under development. If you have any ideas or tips, please feel free to report that at the issue tracker.

* BEM

## Example SCSS structure

```scss
// abstracts/
// - _functions.scss
// - _mixins.scss
// - _variables.scss

// base/
// - _base.scss
// - _fonts.scss
// - _typography.scss
// - _helpers.scss

// components/
// - _button.scss

// layout/
// - _grid.scss
// - _site-header.scss
// - _site-footer.scss

// pages/
// - _home.scss

// vendor/
// - _normalize.scss
```
